# Resources

Curated collection of papers, cheat sheets, roadmaps, and learning materials for Agentic AI.

## üìö Academic Papers

### Foundational Papers
- [ReAct: Synergizing Reasoning and Acting in Language Models](https://arxiv.org/abs/2210.03629) - Introduces reasoning and acting in language models
- [AutoGPT: An Autonomous GPT-4 Experiment](https://github.com/Significant-Gravitas/Auto-GPT) - Open-source autonomous AI agent
- [Toolformer: Language Models Can Teach Themselves to Use Tools](https://arxiv.org/abs/2302.04761) - Self-supervised learning of tool use
- [HuggingGPT: Solving AI Tasks with ChatGPT and its Friends](https://arxiv.org/abs/2303.17580) - Collaborative AI task solving
- [WebGPT: Browser-assisted question-answering with human feedback](https://arxiv.org/abs/2112.09332) - Web-browsing agent architecture
- [Language Models as Zero-Shot Planners](https://arxiv.org/abs/2201.07207) - Planning capabilities in LLMs
- [Constitutional AI: Harmlessness from AI Feedback](https://arxiv.org/abs/2212.08073) - Self-improving AI safety methods

### Multi-Agent Systems
- [Communicative Agents for Software Development](https://arxiv.org/abs/2307.07924) - ChatDev framework for software development
- [MetaGPT: Meta Programming for Multi-Agent Collaborative Framework](https://arxiv.org/abs/2308.00352) - Advanced multi-agent programming
- [AutoGen: Enabling Next-Gen LLM Applications via Multi-Agent Conversation](https://arxiv.org/abs/2308.08155) - Microsoft's multi-agent framework
- [Emergent Communication in Multi-Agent Deep Reinforcement Learning](https://arxiv.org/abs/1706.02275) - Communication protocols in multi-agent RL
- [Cooperative Multi-Agent Learning: The State of the Art](https://arxiv.org/abs/1711.08022) - Survey of cooperative learning methods
- [Multi-Agent Actor-Critic for Mixed Cooperative-Competitive Environments](https://arxiv.org/abs/1706.02275) - MADDPG algorithm

### Planning and Reasoning
- [Chain-of-Thought Prompting Elicits Reasoning in Large Language Models](https://arxiv.org/abs/2201.11903) - Foundation of CoT reasoning
- [Tree of Thoughts: Deliberate Problem Solving with Large Language Models](https://arxiv.org/abs/2305.10601) - Advanced reasoning technique
- [Self-Consistency Improves Chain of Thought Reasoning in Language Models](https://arxiv.org/abs/2203.11171) - Improving reasoning reliability
- [ReWOO: Decoupling Reasoning from Observations for Efficient Augmented Language Models](https://arxiv.org/abs/2305.18323) - Efficient reasoning framework
- [Least-to-Most Prompting Enables Complex Reasoning in Large Language Models](https://arxiv.org/abs/2205.10625) - Decomposed reasoning approach

### Agent Architectures
- [Reflexion: Language Agents with Verbal Reinforcement Learning](https://arxiv.org/abs/2303.11366) - Self-reflective agents
- [REACT: Synergizing Reasoning and Acting in Language Models](https://arxiv.org/abs/2210.03629) - Reasoning and acting integration
- [AgentGPT: Autonomous AI Agents in Web Browsers](https://arxiv.org/abs/2308.10848) - Web-based autonomous agents
- [Cognitive Architectures for Language Agents](https://arxiv.org/abs/2309.02427) - Comprehensive agent design patterns

## üìñ Recommended Books

### Technical Books
- **[Artificial Intelligence: A Modern Approach (4th Edition)](https://www.amazon.com/Artificial-Intelligence-Modern-Approach-4th/dp/0134610997)** by Stuart Russell & Peter Norvig
  - The definitive AI textbook covering agents, search, planning, and reasoning
  - [PDF Preview](http://aima.cs.berkeley.edu/) | [Course Materials](http://aima.cs.berkeley.edu/instructors.html)

- **[Multi-Agent Systems: Algorithmic, Game-Theoretic, and Logical Foundations](https://www.amazon.com/Multi-Agent-Systems-Algorithmic-Game-Theoretic-Foundations/dp/0521899435)** by Yoav Shoham & Kevin Leyton-Brown
  - Comprehensive coverage of multi-agent systems theory and practice
  - [Free Online Version](http://www.masfoundations.org/mas.pdf)

- **[Reinforcement Learning: An Introduction (2nd Edition)](https://www.amazon.com/Reinforcement-Learning-Introduction-Adaptive-Computation/dp/0262039249)** by Richard S. Sutton & Andrew G. Barto
  - Essential foundation for agent learning and decision-making
  - [Free Online Version](http://incompleteideas.net/book/the-book.html)

- **[Deep Learning](https://www.amazon.com/Deep-Learning-Adaptive-Computation-Machine/dp/0262035618)** by Ian Goodfellow, Yoshua Bengio & Aaron Courville
  - Neural network foundations for modern AI agents
  - [Free Online Version](https://www.deeplearningbook.org/)

### Practical Guides
- **[Building LLM Applications for Production](https://www.amazon.com/Building-LLM-Applications-Production-Huyen/dp/1098150961)** by Chip Huyen
  - Practical guide for deploying language model applications
  - [Author's Blog](https://huyenchip.com/blog/)

- **[Hands-On Large Language Models](https://www.amazon.com/Hands-Large-Language-Models-Understanding/dp/1098150952)** by Jay Alammar & Maarten Grootendorst
  - Practical implementation guide for LLM applications
  - [Jay Alammar's Blog](http://jalammar.github.io/)

- **[The AI Engineer's Handbook](https://www.amazon.com/AI-Engineers-Handbook-Building-Applications/dp/1803246618)** by Various Contributors
  - Comprehensive guide for AI application development

- **[LangChain AI Handbook](https://www.pinecone.io/learn/langchain/)** by Pinecone
  - Practical guide to building with LangChain framework
  - [Free Online](https://www.pinecone.io/learn/langchain/)

### Emerging Topics
- **[The AI Revolution: The Road to Superintelligence](https://www.amazon.com/AI-Revolution-Road-Superintelligence/dp/152454394X)** by Louis A. Del Monte
  - Future perspectives on autonomous AI systems

- **[Human Compatible: Artificial Intelligence and the Problem of Control](https://www.amazon.com/Human-Compatible-Artificial-Intelligence-Problem/dp/0525558616)** by Stuart Russell
  - Critical perspectives on AI safety and alignment

- **[The Alignment Problem](https://www.amazon.com/Alignment-Problem-Machine-Learning-Values/dp/0393635821)** by Brian Christian
  - Understanding AI safety and alignment challenges
